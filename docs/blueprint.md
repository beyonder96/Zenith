# **App Name**: Zenith Vision

## Core Features:

- Coordinate Acquisition: Acquire user coordinates upon app launch to provide relevant, location-based services. Request coordinate on launch and store it locally.
- Daily Finance Overview: Display a summary of the day's financial activities. Allows to store finances to the user device
- Shopping List: Create and manage a shopping list, tracking the completion status of each item. Stores lists locally to the device.
- Daily Tasks: Manage a list of daily tasks, allowing users to mark them as complete.
- Intelligent Task Suggestions: Generative AI tool will learn a user's habits from previous days, then proactively suggest daily tasks.

## Style Guidelines:

- Primary color: Vivid pink-purple gradient (#E91E63 to #FF9800) to convey modernity and sophistication.
- Background color: Dark, near-black (#121212) to enhance readability and focus.
- Accent color: Soft coral (#FF7F50) for highlighting key interactive elements.
- Headline font: 'Poppins' sans-serif for a modern and readable look. Body font: 'PT Sans' as body, due to improved readability in longer text
- Clean, minimalist icons for key navigation and action items.
- Card-based layout for clear separation of information and a clean, modern feel.
- Subtle transition animations on content updates to maintain user engagement.
